**Ультимативный набор инструментов для обхода блокировок и снятия замедлений трафика, инициируемых РКН (Роскомнадзором).**

Проект включает три мощные утилиты с единым интерфейсом для борьбы с цензурой, замедлением YouTube и другими видами ограничений в РФ. Работает на Windows 7/10/11.

## 🚀 Что входит в комплект?

| Инструмент | Назначение |
| :--- | :--- |
| **Zapret DPI** | Борьба с DPI (Deep Packet Inspection). Снятие замедлений YouTube, Discord, Telegram и других сервисов. Доп функция FlClashX прокси (Экспериментально) |
| **AdGuard Home** | Сетевой фильтр рекламы, трекеров и блокировка вредоносных доменов на уровне DNS. |
| **TGWsProxy** | Локальный прокси-сервер для стабильной работы Telegram Desktop при проблемах с подключением. |

---

## ✨ Основные возможности

- ✅ **Полный обход DPI** с помощью Zapret (WinDivert-фильтрация).
- ✅ **Автозапуск в качестве служб Windows** – работают в фоне после перезагрузки.
- ✅ **Управление DNS** (CloudFlare, Google, Yandex, AdGuard) через графическое меню.
- ✅ **Гибкая настройка фильтров** – IPSet, игровые порты, пользовательские списки.
- ✅ **Автообновление** списков IP, hosts-файлов и самой утилиты Zapret.
- ✅ **Диагностика системы** – проверка конфликтов с другими сервисами (VPN, Killer, SmartByte).
- ✅ **Мастер установки** – все утилиты работают из одного каталога с правами администратора.
